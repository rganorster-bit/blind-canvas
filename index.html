Create a complete mobile-friendly web game in Hebrew called: "קווים עיוורים" (The Blind Canvas) The game should follow the Funmigo party-game style. GAME FLOW: 1. Display a welcome screen with the game logo and title: "קווים עיוורים" 2. One participant is selected as the artist. 3. The artist presses a button: "פתח משימה" 4. The game randomly selects a drawing challenge from a large collection of at least 100 drawing prompts. 5. The prompt is shown for 5 seconds only. 6. After 5 seconds: - The prompt disappears. - A 60-second countdown timer begins automatically. 7. During the countdown: - The artist is supposed to draw while blindfolded. - The other players cannot see the prompt. 8. When the timer reaches zero: - Display a large button: "הגיע זמן הניחושים" 9. The group tries to guess what was drawn. 10. Add a button: "חשוף תשובה" 11. Reveal the original prompt. 12. Add a button: "משימה חדשה" 13. Randomly select a new challenge. DESIGN: - Hebrew RTL layout. - Funmigo style. - Dark blue background. - Yellow, white and black colors. - Large playful buttons. - Fun animations. - Mobile friendly. - Large countdown timer. - Professional commercial-quality design. DRAWING CHALLENGES: Easy: - שמש - ירח - בית - עץ - פרח - חתול - כלב - דג - ציפור - גלידה - כדורגל - מכונית - אופניים - עפיפון - לב Medium: - טיל ממריא - גמל - טירה - סופרמן - חד קרן - אי בודד - רובוט - ספינת פיראטים - דרקון - ליצן - מגדל אייפל - גן חיות - רכבת הרים - צוללת - טרקטור Funny: - תרנגולת רוקדת - חתול משחק כדורגל - פיל על קורקינט - כריש עם משקפיים - כלב מנגן בגיטרה - קוף אוכל פיצה - סבתא על סקייטבורד - פינגווין גולש - חד קרן עושה סלפי - ג'ירפה במכונית Hard: - פארק שעשועים - חוף ים - חללית בחלל - קרקס - חתונה - בית ספר - מסיבת יום הולדת - שדה תעופה - קמפינג - ג'ונגל Create at least 100 total prompts. Use HTML, CSS and JavaScript in a single file. No external libraries required. Make it feel like a real Funmigo game.
